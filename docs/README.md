## Core Concepts

- **MSF** - The runtime format, fully deterministic and explicit
- **IIS** - Instrument Intent Spec, describes musical goals
- **Inventory** - Canonical source of truth for audio samples
- **Compiler** - Transforms IIS into MSF
- **Runtime** - Plays back MSF instruments from MIDI
